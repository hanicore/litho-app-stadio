## 📖 About

Litho Studio is a modern desktop application that converts ordinary images into high-quality 3D printable lithophanes.

With an intuitive interface and real-time controls, users can generate flat or cylindrical lithophanes, preview the result, and export ready-to-print STL files.

---

## ✨ Features

- 🖼 Convert images into lithophanes
- 🧊 Real-time 3D preview
- 📏 Adjustable dimensions
- 📐 Control minimum and maximum thickness
- 🪄 Image enhancement options
- 🌀 Flat & Cylindrical lithophanes
- 📦 Export printable STL files
- ⚡ Fast generation
