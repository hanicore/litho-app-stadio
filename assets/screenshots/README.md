## 🖥️ Application Interface Preview

Explore the Litho Studio interface and its powerful lithophane creation workflow.

### 🏠 Main Dashboard

The main application window provides a clean and modern workspace where users can import images, adjust lithophane parameters, and control the generation process.

### 🖼️ Image Processing Panel

Users can upload images and customize how the photo is converted into a 3D lithophane model, with options for adjusting dimensions and quality settings.

### 🧊 3D Model Preview

The integrated 3D preview allows users to visualize the generated lithophane model before exporting. Users can inspect the shape, depth, and overall appearance of the model.

### ⚙️ Customization Controls

The settings panel provides controls for:

* Model size and dimensions
* Thickness adjustment
* Lithophane shape selection
* Preview options
* STL export configuration

### 📦 Export Workflow

After creating the model, users can export the final result as an STL file, ready for 3D printing.
